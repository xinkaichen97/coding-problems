# Coding Problems
 
Python implementations with comments and/or notes for a variety of coding problems, covering data structures, algorithms, and machine learning fundamentals.
 
## Problem Sources
 
- [Leetcode](https://leetcode.com/) — Classic algorithm and data structure problems, widely used for software engineering interview prep.
- [Neetcode](https://neetcode.io/) — Curated problem lists and video explanations organized by topic (e.g., arrays, graphs, dynamic programming).
- [DeepML](https://www.deep-ml.com/) — Machine learning-focused coding challenges covering topics like linear algebra, probability, and ML algorithms from scratch.
 
## Structure
 
Each solution includes inline comments explaining the approach, time/space complexity notes where relevant, and occasionally alternative solutions or edge case discussions.
